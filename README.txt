Pour pouvoir lancer l'application :
- Importer la base de donnée sql dans phpmyadmin
- Placer tous les fichiers dans C:\wamp64\www\'nom_du_dossier'
- Lancer wamp 

Pour lancer l'application en tant que client entrer cet url :
- http://localhost/'nom_du_dossier'/Controler.php

Pour lancer l'application en tant qu'admin entrer cet url :
- http://localhost/'nom_du_dossier'/DBControler.php
